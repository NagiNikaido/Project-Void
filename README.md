Project-Void
================

***Project-Void***，一个（将来也许能够）用来代替Lemon、Cena、……的评测系统。暂时只具备在Linux下对c++测一道题的能力。（请不要吐槽这些）

实现基于python和xml，python2.7.6和python3.4.0均能正常运行。
